What is the point of this program?#This program is intended to help medical practices efficiently and effectively complete their administration work. It does this by giving medical practices the ability to keep patient records and files digitally, as well as by providing a digital booking function to make organization easy. The program also allows doctors to issue pharmaceutical scripts to patients digitally while also recording it in the practice's own records for reference purposes. 
How do I use this program?#Each screen and interface in this program has been labelled for an easy user experience. Read the labels and follow the logical flow of the program.
